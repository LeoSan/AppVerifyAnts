### Tecnologias aplicadas 
#### Desarrollo Back 
- Node.Js V15-14 
- Expresss 
- Mongo DB
- Mongoose 

#### Comandos Back 
- npm init
- npm i express
- npm i --save-dev nodemon
- npm install mongoose --save
- npm install dotenv
- npm run dev -> Claro debes configurar ciertos archivos
